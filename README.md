# Z-Chat Tasker 🚀

A lightweight, efficient task management system built with Python using **Socket Programming** and **JSON** for communication.

## ✨ Features
- **Client-Server Architecture:** Decoupled communication using TCP Sockets.
- **Graphical User Interface (GUI):** Built with `PySimpleGUI` for an intuitive user experience.
- **Data Exchange:** Uses `JSON` for structured and reliable data transmission.
- **Real-time Dashboard:** View all tasks stored on the server instantly.
- **Linux Optimized:** Tested and deployed on Rocky Linux.

## 🛠️ Requirements
- Python 3.12+
- `pysimplegui` library
- `tkinter` (System level package)
